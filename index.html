Amazon Connect CTI Adapter Salesforce Package

The newly updated Amazon Connect CTI Adapter v5 makes it easy to use your Amazon Connect contact center with Salesforce to deliver engaging service with lower cost at any scale. Amazon Connect is cloud-based, self-service, and can be set up in minutes.

Amazon Connect Salesforce Lambdas

The AWS Serverless application package contains a set of common Lambda functions to be used by Amazon Connect to interact with Salesforce, allowing lookup, create and update operations for different Salesforce objects, like Contacts and Cases.

Deep Integration between Amazon Connect and Salesforce

With those components, customers can build a deep integration between the Amazon Connect contact center platform and Salesforce, the leading customer relationship management (CRM) platform.
